## GIT Commit Message rule



| Tag Name | Description |
| --- | --- |
| feat | 새로운 기능추가 |
| fix | 버그 수정 |
| design | CSS 및 UI 디자인 변경 |
| refactor | 코드 리팩토 |
| comment | 주석 추가 및 변경 |
| docs | 문서 수정 |
| test | 테스트 추가 |
| chore | 빌드 테스트 업데이트, 패키지 매니저를 설정하는 경우 |
| rename | 폴더명 변경 |
| remove | 파일을 삭제 시 |
